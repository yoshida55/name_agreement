# CSS枠生成インストラクション（レベル2：標準版）

## 目的
HTMLコードとスクリーンショットから、構造が分かりやすいCSS枠を生成する。
Flexboxを使う箇所を明確にし、コメントで階層構造を可視化する。

---

## インプット（必須）

1. **HTMLコード**（セクション単位）
2. **スクリーンショット**（ブラウザ表示）

※既存のCSS命名規則がある場合は、それに従う

---

## アウトプット

### 1. CSS枠（コメント付き）
- セクションごとに区切りコメント
- 階層構造が分かるインデント
- Flex親子を明示

### 2. 構造の解説
- Flexboxの使用箇所と理由
- Flexboxを使わない箇所と理由
- 階層構造の説明
- 視覚的な構造図（テキストベース）

### 3. 💡 実装のコツ（Notion向けフォーマット）
- Flex親の具体的な書き方例
- Flex子要素の具体的な書き方例
- よく使うプロパティのヒント
- **フォーマット要件**：
  - 見出しとコードブロックを明確に分ける
  - 各プロパティに簡潔な説明を付ける
  - Notionで見やすい形式で出力

### 4. SVG構造図（確認後に生成）
- CSS枠・構造解説・実装のコツを出力後、「**SVG構造図が必要ですか？**」と確認
- 必要な場合のみ、構造を視覚化したSVG画像を生成
- 文字サイズは大きめに設定（Notionでの視認性を考慮）

---

## CSS枠の書き方ルール

### セクション区切りコメント
各セクションの最初に、枠で囲んだコメントをつける：

```css
/* ┌─────────────────────────────────────────┐
   │ セクション名（説明）                     │
   └─────────────────────────────────────────┘ */
```

### 階層構造のコメント記法

| 記号 | 意味 | 使う場面 |
|------|------|----------|
| （コメントなし） | 親要素 | セクションのトップレベル |
| `/*   └ 子要素 */` | 通常の子要素 | 親の直下、Flexじゃない |
| `/*   ▼ Flex親 */` | Flex親要素 | `display: flex` を使う |
| `/*     └ Flex子要素 */` | Flex子要素 | Flex親の直下 |

### インデントルール
- 親要素：インデントなし
- 子要素：2スペース
- Flex子要素：4スペース
- さらに深い階層：2スペースずつ追加

---

## Flexbox使用の判断基準

### ✅ Flexboxを使う場合
- **横並び**が必要
- **均等配置**が必要
- **左右に分ける**配置（ヘッダーなど）
- **カードレイアウト**（複数枚を横並び）

### ❌ Flexboxを使わない場合
- **縦並び**だけ
- **単純なコンテナ**（中身を包むだけ）
- **テキスト表示**のみ

## 生成フロー

```
1. HTMLとスクショを受け取る
   ↓
2. 構造を分析
   - どこが親子関係か
   - どこでFlexを使うか
   ↓
3. CSS枠を生成
   - セクション区切りコメント
   - 階層コメント
   - 空の{}ブロック
   ↓
4. 構造の解説を出力
   - Flexboxの使用箇所と理由
   - Flexboxを使わない箇所と理由
   - 階層構造の説明
   ↓
5. 💡 実装のコツを出力
   - Notion向けの見やすいフォーマット
   - 具体的なCSSコード例
   - プロパティの説明
   ↓
6. 「SVG構造図が必要ですか？」と確認
   ↓
7. 必要な場合のみSVG生成
```

## CSS枠の基本フォーマット

```css
/* ===============================================
  セクション大分類
 =============================================== */

/* ┌─────────────────────────────────────────┐
   │ セクション名（説明）                     │
   └─────────────────────────────────────────┘ */
.parent-class {
    
}

/*   └ 子要素 */
.child-class {
    
}

/*   ▼ Flex親（説明） */
.flex-parent-class {
    
}

/*     └ Flex子要素 */
.flex-child-class {
    
}

/*       └ さらに深い階層 */
.nested-class {
    
}
```

## 💡 実装のコツ

### 1️⃣ `.クラス名`（Flex親）

​```css
display: flex;
gap: 20px;
justify-content: center;
​```

- `display: flex` → 横並びにする
- `gap: 20px` → 要素間の余白
- `justify-content: center` → 中央寄せ

---

### 2️⃣ `.クラス名`（Flex子要素）

​```css
flex: 1;
max-width: 300px;
​```

- `flex: 1` → 均等幅にする
- `max-width: 300px` → 最大幅を制限

---

### 3️⃣ その他のよく使うプロパティ

​```css
.クラス名 img {
    width: 100%;
    height: auto;
    object-fit: cover;
}
​```

- `width: 100%` → 親要素の幅に合わせる
- `height: auto` → 縦横比を維持
- `object-fit: cover` → 切り抜いて表示
```

**重要ポイント**：
- 見出しレベルを適切に使い分ける
- コードブロックと説明を明確に分ける
- 各プロパティに1行で説明を付ける
- 絵文字を使って視認性を高める

---

## 注意事項

1. **空のブロックを作る**
   - 中身は書かない
   - `{ }` の間に1行空ける

2. **クラス名はHTMLから抽出**
   - 変更しない
   - そのまま使う

3. **コメントは構造だけ**
   - 実装方法は書かない
   - 階層関係を示すだけ

4. **Flex判断を明確に**
   - Flex親には必ず `▼` をつける
   - Flex子要素には必ず `└ Flex子要素` と書く
